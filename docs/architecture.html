github.com/jiujuan/goagent · Layered Architecture

goagent 分层架构

一个模块化、可扩展的纯 Go Agent 框架，融合 pi / adk-go / langchaingo 三者精华。零外部依赖，全栈以统一事件流贯穿。

一切皆事件流 · 事件即数据 · 能力即中间件 · 接口尽量小

Go 1.23+ · range-over-func 零依赖 · 纯标准库 流式原语 iter.Seq2[*Event,error] 配置 Functional Options 持久化 JSONL append-only 协议 Anthropic / OpenAI / MCP

L4 编排层 · Orchestration runner/ · plan/ 顶层入口：解析会话、提交事件、运行 root agent、持久化、后台异步与 DAG 计划执行
Runner
runner/runner.go
编排入口。解析会话 → 提交 user 事件 → 运行 root agent → 把非 partial 事件持久化到 Store。
Background
runner/background.go
把一次 agent run 封装为 queue.Job，桥接到工作池，实现 fire-and-forget 异步执行。
Plan 引擎
plan/
DAG 执行计划：Plan/Step 模型 + 依赖感知并发 Scheduler + Executor（工具/agent/函数）+ 重规划/审批/快照恢复。
SchedulerExecutorErrorPolicySnapshot
L3 决策层 · Agent agent/ Agent 契约与实现：LLM 驱动的 turn 引擎、确定性编排、流水线、委派与多模态 agent
LLMAgent
agent.go
核心 turn 引擎：model↔tool 循环（maxSteps=16），装配 middleware chain。
Workflow
workflow.go
确定性编排：Sequential 顺序 / Loop 循环(Escalate跳出) / Parallel 并发(分支隔离)。
Pipeline
pipeline.go
Fluent 声明式流水线 Pipeline().Then().ThenParallel().ThenLoop().Build()。
Transfer
transfer.go
LLM 驱动委派：注入 transfer_to_agent 工具 + 方向规则(sub/parent/peer) + 防环深度保护。
ImageAgent
imageagent.go
包装 llm.ImageModel 的图片生成 agent。
VideoAgent
videoagent.go
包装 llm.VideoModel 的视频生成 agent（异步轮询）。
L2 模型调用层 · Middleware middleware/ 能力即中间件 — Middleware = func(llm.Model) llm.Model，层层装饰模型，互不耦合
Chain 组装顺序（外 → 内）
RateLimit → Compaction → RAG → HITL → Retry → Steering → bare Model
Compaction
历史过长时自动用 LLM 生成结构化摘要替换旧消息（上下文压缩）。
Retry
指数退避重试，仅在未产出 token 前重试，避免重复输出。
RateLimit
令牌桶限流 + 并发数控制。
HumanInTheLoop
危险工具人工审批门禁：approve / deny / editArgs。
Steering
运行中跨 goroutine 注入消息，实时"插话"引导。
Resummarize
持久化摘要，写入会话树 SummarizesTo 节点。
L1 原语层 · Primitives core/ · llm/ · tool/ 共享词汇表 — 定义全栈数据模型与最小接口，打破 import 循环，不依赖任何上层
core
core/ · 共享 schema
Message / Part(sealed union) 多模态消息；Event + Actions 流单元；Stream=iter.Seq2 全栈流式原语；NewID 随机 ID。
MessageEventStreamPart
llm
llm/ · 模型接口
Model / ImageModel / VideoModel 统一抽象 + Functional Options。Provider：anthropic 原生、openaicompat 兼容、agnes 图/视频、mock。
anthropicopenaicompatagnesmock
tool
tool/ · 工具契约
Tool 小接口 + 泛型 Func[In,Out]，用反射自动生成 JSON Schema。内建：exec 沙箱命令、web 联网搜索、mcp 外部工具。
Func[In,Out]execwebmcp
CC 横切关注点 · Cross-Cutting 贯穿各层的可插拔基础服务
session
会话持久化：事件树(ParentID)、分支/fork、JSONL 日志、InMemory/FileStore。
memory
长期记忆/RAG：向量库 + 余弦检索 + 检索注入中间件 + search_memory 工具。
prompt
可组合 System Prompt：Section(Name/Order/Render) + Builder 按序拼接。
sandbox
受控命令执行：五项限制(目录/超时/输出/白名单/环境)，跨平台进程后端。
queue
后台执行：内存队列 + 发布订阅 Bus + 有界并发 Worker 池。
skill
文件系统技能包：三层渐进加载(元数据→body→脚本)，基于 fs.FS。
embeddings
文本向量化抽象 Embedder：mock + OpenAI 兼容。
callbacks
可观测性 Handler：7 个 On* hook，零开销 Noop。
memory/workingmem
结构化 scratchpad(Goal/Todos/Notes)，穿越压缩不丢失。
memory/textmem · projectmem · rules
Markdown 事实库 / AGENTS.md / 两级规则注入。
internal/sse
最小 SSE 解析器，anthropic 与 openaicompat 共用。
